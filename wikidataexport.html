Wikidata export
With this node you can upload data to Wikidata. You must first see if the items you plan to add or link to exist already in Wikidata using Wikidata reconcile node.
Choose the GLAMpipe field which has Wikidata QIDs for the items already in Wikidata and empty values for items that don't exist yet.
